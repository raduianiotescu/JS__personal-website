Lighthouse Accessibility Audit

By Radu Ianiotescu   |   August 4, 2025

Making websites accessible isn’t just about compliance, it’s about creating inclusive experiences for everyone. A powerful and simple way to get started is by using Lighthouse, a free auditing tool built right into Chrome. In this article, I’ll show you exactly how to use Lighthouse in DevTools to generate an accessibility report, and encourage you to run it on this very site!

What is Lighthouse?

Lighthouse is an open-source tool developed by Google that analyzes web pages and provides performance, SEO, best practices, and, most importantly, accessibility scores. It gives you a clear, actionable breakdown of how your site performs and what can be improved.

How to Open Lighthouse in DevTools

To run Lighthouse from Chrome:

  1. Open the website you want to audit, try it on this site if you’d like!
  2. Right-click anywhere on the page and select Inspect to open Chrome DevTools. You can also press Ctrl+Shift+I (or Cmd+Option+I on Mac).
  3. In DevTools, find the Lighthouse tab. If you don’t see it, click the double arrow (») at the top to expand more tabs.

Setting Up the Audit

Once you’re in the Lighthouse tab:

  • Under Categories, make sure only Accessibility is checked.
  • Choose the device type: Mobile or Desktop. Desktop is fine for now.
  • Click the blue Analyze page load button to start the audit.

Chrome will reload the page, run the audit, and display a detailed report within a few seconds.

Understanding the Report

At the top of your report, you’ll see an Accessibility score from 0 to 100. Below that, Lighthouse lists specific issues found on the page. Some common items might include:

  • Images without descriptive alt text
  • Low contrast between text and background
  • Buttons or links missing accessible names
  • Elements not reachable by keyboard navigation

Each issue comes with an explanation and suggestions on how to fix it, including links to documentation for deeper learning.

Try It Yourself

Open DevTools right now and run a Lighthouse audit on this page. See how it performs! Then, try it on a few other sites you visit regularly. You’ll be surprised how many small issues can add up, and how easy they are to start fixing.

Accessibility doesn’t have to be intimidating. With tools like Lighthouse, it’s easy to identify opportunities for improvement and build more inclusive websites, one audit at a time.
